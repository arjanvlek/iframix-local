# Static Webapp asset files used per iFramix app version

MOST RECENT VERSION: <NONE_YET>

The iFramix Pro webapp is a Vue 2 SPA built with Vite. Its asset files are content-hashed,
so every new app version ships a different set of hashed CSS / JS / font bundles. This
file maps every minified asset file under `webapp/static/` to the app version it belongs
to, so that future versions can be added alongside (or in place of) the previous version' files
without losing track of which file came from which build.
